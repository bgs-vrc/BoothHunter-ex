# BoothHunter-ex

본 프로젝트는 [eddy0626/BoothHunter](https://github.com/eddy0626/BoothHunter)에서 포크하여 기능을 추가한 버전입니다. 즐겨찾기 기능을 확장하여 구매한 에셋에 대한 저장과 관리에 대한 편의성 제공이 주 목표입니다.

## 추가 기능
* **컬렉션 태그 자동 추가**: 에셋을 즐겨찾기 할 때 사전에 지정해둔 태그와 일치하면 자동으로 지정 컬렉션에 추가됩니다.
* **데이터 백업 및 복원 (Export/Import)**: 즐겨찾기 항목 및 컬렉션 자동 추가 규칙을 로컬 파일(.txt) 단위로 내보내고 불러올 수 있습니다.
* **로컬 에셋 폴더 연동**: 데스크탑 디스크 경로에 에셋별 단독 폴더를 생성하고 바로 열어볼 수 있도록 지원합니다.
* **데이터베이스 위치 커스텀**: 기본 DB 경로 대신 사용자가 원하는 저장 경로로 앱 DB 파일을 자유롭게 변경할 수 있습니다.
